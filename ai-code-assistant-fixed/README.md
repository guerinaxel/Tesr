# AI Code Assistant (Improved)

This repository contains an improved version of the AI code assistant:

- **backend/** – Django 5 + DRF + RAG (FAISS + SentenceTransformers) + Ollama integration.
- **frontend/** – Angular standalone chat UI consuming `/api/code-qa/`.
- **docker-compose.yml** – Postgres + Ollama + backend.

See `backend/README-backend.md` and `frontend/README-frontend.md` for more details.
